## Quick Start

Install Node.js and then:

```sh
$ git clone the project
$ cd into the project
$ sudo npm -g install grunt-cli karma bower
$ npm install
$ bower install
$ grunt watch
$ open a new terminal session
$ cd into the project
$ sudo npm -g install http-server
$ http-server build
```

Finally, open `localhost:8080` in your browser.
